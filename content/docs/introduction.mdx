# Welcome to Elide

Elide is a modern, high-performance runtime designed for building scalable applications. It combines the best features of multiple programming languages and runtimes into a unified platform.

<Alert>
  <strong>Getting Started:</strong> This documentation will guide you through everything you need to know about Elide.
</Alert>

## What is Elide?

Elide is a polyglot runtime that supports multiple programming languages including JavaScript, TypeScript, Python, and more. It provides:

- **High Performance**: Built on GraalVM for optimal execution speed
- **Polyglot Support**: Run multiple languages in the same application
- **Modern APIs**: Clean, intuitive APIs for common tasks
- **Security First**: Built-in security features and sandboxing

## Why Choose Elide?

Traditional runtimes force you to choose between performance, developer experience, and language flexibility. Elide provides all three:

1. **Performance**: Native compilation and advanced optimizations
2. **Flexibility**: Use the best language for each task
3. **Simplicity**: Clean APIs and excellent documentation

## Use Cases

Elide is perfect for:

- **Web Applications**: Build fast, responsive web apps
- **APIs and Microservices**: Create efficient backend services
- **Serverless Functions**: Deploy with minimal cold start times
- **CLI Tools**: Build fast command-line applications
