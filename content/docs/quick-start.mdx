# Quick Start

This guide will help you create your first Elide application in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18 or later
- npm, yarn, or pnpm
- A code editor (VS Code recommended)

## Installation

Install the Elide CLI globally:

## Create Your First Project

Use the CLI to create a new project:

```bash
elide create my-app
cd my-app
```

This will create a new directory with the following structure:

```
my-app/
├── src/
│   ├── index.ts
│   └── app.ts
├── elide.config.js
└── package.json
```

## Project Structure

Let's look at the main files:

### `src/index.ts`

This is your application's entry point:

```typescript
import { serve } from '@elide/server'

serve({
  port: 3000,
  handler: (req) => {
    return new Response('Hello from Elide!')
  }
})
```

### `elide.config.js`

Configuration file for your Elide application:

```javascript
export default {
  runtime: {
    engine: 'v8',
    optimize: true
  }
}
```

## Run Your Application

Start the development server:

```bash
elide dev
```

Your application will be available at `http://localhost:3000`

<Alert>
  <strong>Hot Reload:</strong> The dev server automatically reloads when you make changes to your code.
</Alert>

## Build for Production

When you're ready to deploy, create a production build:

```bash
elide build
```

This creates an optimized binary in the `dist/` directory.
